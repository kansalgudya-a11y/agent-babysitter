## Before you write in

A quick self-check solves most reports:

- **Update first.** Use License & Updates → check for updates and install the
  latest version. The daily update check contacts `api.github.com` only.
- **"An app whose developer cannot be verified" / Gatekeeper.** Beta builds may
  be unsigned. Open the app once, then go to System Settings → Privacy &
  Security → "Open Anyway". Instructions also ship inside the DMG.
- **An agent isn't showing up.** The app auto-detects installed agents from
  their own files. Make sure the agent has actually run at least once on this
  Mac so its files exist (for example `~/.claude/projects/` or
  `~/.codex/sessions/`).
- **Cost or limit shows "—".** Some surfaces genuinely can't report a number.
  Antigravity is activity-based (no public schema), so it shows Working / Done /
  Ended and no cost. Codex and Antigravity read usage locally; Claude, Cursor,
  and Manus percentages require the opt-in **Live usage** toggle (Settings →
  Advanced), which is off by default.
- **Notifications aren't appearing.** Check macOS System Settings →
  Notifications for the app, and confirm quiet hours / mute aren't on in
  Preferences. Exact "waiting/done" banners for Claude Code need Precision mode.

## Reporting a bug well

Please include:

1. App version (from the menu or License & Updates).
2. macOS version and Mac model (Apple menu →  About This Mac).
3. Which agent(s) are involved (Claude Code, Codex, Antigravity, Gemini, Cursor,
   Manus) and how they were launched (CLI or desktop app).
4. What you expected vs. what you saw. Screenshots of the menu help.
5. Whether any opt-in feature was on (Live usage, Precision mode, iCloud stats
   sync, Claude usage meter).

**Please do not send us your prompts, transcripts, or source code.** The app is
built so that data stays on your Mac (see PRIVACY.md), and we don't need it to
diagnose most issues.
